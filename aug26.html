JavaScript String Methods:

document.write method:

document.write method is used to print anything in a web page.
Example:document.write("Welcome to MITS WEBCLUB.");
Output:Welcome to MITSWEBCLUB.

length() method:

Length() method is used to find the length of the string. Example:My name is Jyothsna.
x.length()=20

indexOf():

The indexOf() method returns the index of the first occurrence of a specified text.
Example:var x="Hello joshna, this is also joshna."
x.indexOf("joshna",5)=6

lastIndexOf() method:

The lastIndexOf() method returns the index of the last occurrance of a specified text in a output.
Example:var x="Hello joshna, this is also joshna."
x.lastIndexOf("joshna")=27

search() method:

The search() method searches a string for a specified value and returns the position of the value.But it will not accepct the second parameter of indexOf() method.It has only one parameter.
Example:var x="If you can do it,you can do it."
x.search("you")=3

toLowerCase() method:

The toLowerCase() method is to convert the string to lowercase letters.
Example:var x="ITS OK NOT TO BE OK."
x.toLowerCase()=Its ok not to be ok.

toUpperCase() method:

The toUpperCase() method is to convert the string to lowercase letters.
Example:var x="Its ok NOT to be ok."
x.toUpperCase()=ITS OK NOT TO BE OK.

slice() method:

The slice() extracts a part of a string and returns the extracted part in a new string.It accepts both positive and negative indexes.
Example:var x="Its ok not to be ok."
x.slice(7,13)=not to
x.slice(-7,-1)=be ok

substring() method:

The substring() method is similar to slice() method but it cannot accept negative indexes.
Example:var x="Its ok not to be ok."
x.substring(7,16)=not to be

substr() method:

The substr() method is similar to slice() method but the second parameter specifies the length of the text to be displayed.
Example:var x="Its ok not to be ok."
x.substr(7,6)=not to

replace() method:

The replace() method is used to replace a specified word with another word.It will have two parameters.First parameter is the word which we want to replace and second parameter is a replaced word.This method is case sensitive.
Example:var x="Its ok not to be ok."
x.replace("ok","OK")=Its OK not to be ok

replace using "i" and "g" method:

To replace case insensitive, use a regular expression with an /i flag (insensitive).To replace all matches, use a regular expression with a /g flag (global match).
Example:var x="Its ok not to be ok."
x.replace(/not/i,"NOT")=Its ok NOT to be ok.
x.replace(/not/g,"NOT")=Its ok NOT to be ok.
